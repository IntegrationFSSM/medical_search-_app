DEMARRAGE RAPIDE (3 ETAPES)
================================================================================

ETAPE 1 : Verifier la configuration
------------------------------------
Le fichier .env est deja configure avec votre cle API OpenAI.

IMPORTANT : Modifiez le chemin EMBEDDINGS_FOLDER dans .env si necessaire :
- Ouvrez le fichier .env
- Ligne 7 : EMBEDDINGS_FOLDER=votre/chemin/vers/embeddings
- Sauvegardez

ETAPE 2 : Lancer l'application
-------------------------------
METHODE SIMPLE :
  Double-cliquez sur : start.bat

OU METHODE MANUELLE :
  1. Ouvrez PowerShell/CMD dans ce dossier
  2. Tapez : venv\Scripts\activate
  3. Tapez : python manage.py runserver

ETAPE 3 : Acceder a l'application
----------------------------------
  Ouvrez votre navigateur : http://127.0.0.1:8000/
